Grant and revoke fee allowance for a grantee by a granter

Usage:
  gaiad tx feegrant [flags]
  gaiad tx feegrant [command]

Available Commands:
  grant       Grant Fee allowance to an address
  revoke      revoke fee-grant

Flags:
  -h, --help   help for feegrant

Global Flags:
      --chain-id string     The network chain ID
      --home string         directory for config and data (default "/Users/user/.gaia")
      --log_format string   The logging format (json|plain) (default "plain")
      --log_level string    The logging level (trace|debug|info|warn|error|fatal|panic) (default "info")
      --trace               print out full stack trace on errors

Use "gaiad tx feegrant [command] --help" for more information about a command.
